# METIS Pipeline
The METIS pipeline is the data reduction software for the Mid-infrared E-ELT Imager and Spectrograph (METIS). 

## Installing the software

The software can be installed and run in several ways. 

To run using a docker file, follow the instructions [here](../README.md)

To install and run manually, follow the instructions [here](https://github.com/AstarVienna/METIS_Pipeline/blob/karr/quickFixInstructions/metisp/README.md))
