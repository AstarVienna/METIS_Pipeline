# METIS Pipeline
The METIS pipeline is the data reduction software for the Mid-infrared E-ELT Imager and Spectrograph (METIS). 

## Installing the software

The software can be installed and run in several ways. 

To run using a docker file, follow the instructions [here](metisp/README.md)

To install and run manually, follow the instructions [here](toolbox/README.md))

For basic instructions on running recipes and workflows go [here](metisp/README.md#running-the-software)
